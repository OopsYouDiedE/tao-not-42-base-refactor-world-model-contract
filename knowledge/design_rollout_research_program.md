## R1 — 训推 rollout 配置锁定扫描(TwoTower 教训移植)

- **假设**:生成质量在(训练步长配置 d)×(推理步数 K、rollout 长度 H)不匹配时急剧崩溃
  (TwoTower 实测:块长超训练配置 76.4→19.85),存在须文档化的"安全域"。
- **步骤**:全 token 动力学训练配置 d∈{1/4, 1/8};推理扫 K∈{1,2,4,8} × H∈{4,8,15,30},
  每格测 holdout PSNR@k、判别器分数、N 样本方差。
- **判据**:画出崩溃边界;想象训练与 planner 使用的 (K, H) 必须落在安全域内,
  写入 dataset/训练契约(推理期禁止随手加长)。
- **成本**:推理 only,数小时 L4。依赖 Phase 1。

## R2 — 采样发散度 = 不确定性探针(K 头集成的替代)

- **假设**:同一起点 N 个采样 rollout 的逐步发散度(token 空间方差)与真实开环误差正相关,
  可作模型不确定性信号(承接 C_real 深度校准失败后的"K 头集成分歧"备胎条款——
  流匹配下 N 采样分歧是零额外参数的等价物)。
- **步骤**:R2a(立即,Crafter/dreamerv3):RSSM 想象 N=8 采样,发散度 vs holdout 开环误差
  的 Spearman;R2b(Phase 1 后,gaming500 全 token 动力学同口径,逐游戏分层)。
- **判据**:Spearman ≥ 0.5 采纳为想象截断/降权信号;< 0.3 弃用,回退 K 头集成。
- **成本**:R2a 当天可出;R2b 数小时。
- **状态(2026-07-03)**:**R2a PASS**——60k/62k 两次独立复测 mean per-k ρ = 0.52 / 0.57,
  有效 k 区间从 4k 时的 k≤3 扩到全深度 k=1..15;R3 解锁。
  见 [conclusion_r2a_divergence_probe.md]。

## R3 — 不确定性定价的自适应想象视野

- **假设**:在发散度超阈值处截断想象(或按置信度降权 λ-return 目标),actor 不再从
  "模型已经不知道自己在说什么"的尾部学习,样本效率提升。
- **步骤**:Crafter 线机制验证,同预算三臂——A 固定 H=15 / B 发散度截断 / C 置信降权;
  比成就分数-环境步曲线。
- **判据**:B 或 C 同预算成就分 ≥ A+10%,或同分省 ≥30% 环境步 → 采纳进主线;
  两臂均不显著 → 记录负结果,固定 H 不再纠结。
- **成本**:1–2 天 L4。依赖 R2a 过闸。

## R4 — rollout 难负样本回填 C_real(闸门结论的执行项)

- **假设**:C_real 一步饱和(0.92 vs 0.055)因负样本过易;用安全域内多步自生成状态
  (k=1..8)作负样本,深度校准可恢复。
- **步骤**:R1 安全域内生成负样本库,重训 C_real 探针,复测分数-深度曲线与 flip/swap AUC。
- **判据**:score 对 k 单调递减(Spearman 显著为负)且 flip/swap AUC > 0.6
  (对照原探针的 0.512/0.523);不达标则表征定罪流程重启。
- **成本**:探针级,小时。依赖 R1。

## R5 — 梦境送审试点(L3 的前置闸门)

- **假设**:VLM 对"解码后的想象未来"的合理性/达成判定,与该未来的真实接续
  (hindsight 真值)一致——这是 L3 双向闭环开工的必要条件。
- **步骤**:采样 rollout → 解码 4–8 帧条(唯一解码点)→ Omni(OpenRouter 免费档)
  目标条件化判定"此计划会成功吗";与 holdout 真实接续对照;交换顺序消位置偏置。
- **判据**:VLM 判定与真值一致率显著高于随机且 ≥ 人工抽检一致率的 80% → 开 L3 工程;
  不达标 → L3 冻结,双系统停在 L1/L2。
- **成本**:API 试点数十美元内。依赖解码头(已近完成)+ R1。

## R6 — 冻结塔 + 增量塔的快速适应 rollout(TwoTower 配方移植,北极星主证据)

- **前置 R6a(层可读性探针)**:线性探针跨游戏读冻结通用动力学塔中层——
  分布位移下层对齐是否仍可读(TwoTower 消融 Table 2:冻结+独立塔胜联合微调 26%,
  但其两塔同分布;跨游戏是我们的新增风险点)。可读性差 → R6 降级为 LoRA 微调对照。
- **假设**:留一游戏(leave-one-game-out)设定下,冻结其余游戏训的动力学塔 +
  层对齐 cross-attention 增量塔,能以远小于全微调的数据/步数达到其 rollout 质量
  ——即"换游戏只重学物理增量"的直接证据。
- **步骤**:holdout 游戏仅给 1h 数据;四臂——增量塔 / 全微调 / 从零 / 冻结不加塔;
  测 EV@k 达标所需步数与数据量。
- **判据**:增量塔以 ≤25% 全微调训练量达到其 ≥90% EV@k → 采纳为主线适应机制,
  并作 mental_world 适应元目标的进展证据(注意:此为物理通道,不受语义防火墙约束)。
- **成本**:最大项,Phase 1 后数周。
