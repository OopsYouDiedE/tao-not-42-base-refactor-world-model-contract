# 北极星路线图：多样化成就 · 铁与钻石 · 进入下界（· 刷铁机）

登记日 2026-07-08，同日按用户路线知识修订 v2。终点由用户定义：(A) 多样化成就；(B) 铁与钻石获取；(C) 进入下界；
(D) 终极延伸=指导模型建成刷铁机（用户裁定：视作超越下界的终极成就，等价于具备通关 MC 能力）。
成功口径沿用 M-IRON 宪法：自然 DEFAULT 世界、随机 seed、冷启动、无教师无特权、结果导向（任何路线均可）、n≥16 报成功率+CI+中位耗时。

## 0. 总判决：可达，但不匀速

三个终点均判**可达**，无需推翻现有体系（双塔+宏路线+GRPO 过程优势）。
**v2 关键重排（用户路线知识）**：下界与钻石**解耦**——最容易的下界路径=铁桶+岩浆浇铸速搭传送门
（只依赖 3 铁锭），不需要钻镐。铁的最优路径=村庄铁傀儡（垫高三格安全击杀），不是挖矿。
由此，方块放置技能从 R3 提前到 R1，且用户三条路线天然构成**放置技能课程阶梯**：
垫高三格（1 类方块、自脚下）→ 桶浇铸（精确序列、流体交互）→ 摆门点火 → 刷铁机（蓝图级建造）。
单机墙钟是全程的税：episode 越深吞吐越低（环境瓶颈机器）。

| 贯穿 | M-ACH 多样化成就 | GRPO 里程碑深度分扩展为成就集合 + AD 式多样性奖励 | 成就数曲线单调上升 | 低风险（Crafter PPO+AD 已验证同构方案） |

路线要点：
- **铁的主路线改为村庄铁傀儡**（用户裁定为最优）：寻村→引傀儡→自脚下垫高三格→安全击杀→稳定 3–5 锭。
  需要的放置技能是全域最简形态（跳放 1 类方块），恰好当放置课程入门。挖矿降为辅路线，仍进 GRPO 供学生自选。
- **下界主路线改为桶浇铸**（用户裁定为最容易）：铁桶→找地表岩浆池→水桶交替舀浇原位铸黑曜石逐块搭框→
  点燃跨门。只依赖铁，**钻石从下界依赖链中移除**。浇铸是精确放置序列但完全确定性→宏优先+回读确认。
  备选：废弃传送门捡漏、钻镐正统路线，三路线都进 GRPO。
- **钻石独立成端点**（用户仍要求获取），与下界并行推进：阶梯下挖+条带采掘，唯一硬依赖是岩浆急停。
  洞穴路线留作 RL 自发发现的加分项。
- **成就多样化**是主链副产品优先，AD 多样性 bonus 只负责把"顺路"变成"主动收集"。

### 1.1 M-IRONFARM 可达性初判（终极成就，诚实评估）

刷铁机=村民≥3+床+工作站+惊吓源（僵尸/掉落）+刷出平台+水流汇集+击杀/收集系统。对模型意味着：
读蓝图→选址→数十方块多类放置→实体运输（船/矿车推村民）→系统级验收（傀儡刷出且被收集）。
判**有条件可达**：放置课程走完 R1→R2a 后，蓝图建造=宏骨架（蓝图=相对坐标放置序列，慢塔监工逐段验收）
+ RL 只负责选址与纠错。最大不确定性是**村民运输**（实体交互非确定性，全链最难宏化的一环）。
定位为 R4 北极星延伸：不设时限承诺，R2a 跨门后做专项可行性 spike 再裁决。

## 2. 教师角色扩展：判优 + 文本路线提示（用户指令）

教师从纯裁判扩为"裁判+领航员"，三层实现：

1. **塔内提示（推理时合法）**：慢塔本来就是路线提示器——文本计划→向量/索引下行喂快头（§6.2 契约）。
   强化方向：配方卡式提示模板（E2 已验证 0.73）+ 基于地图模块的"未探索扇区"指路。这部分**不违反冷启动**，
   慢塔是学生身体的一部分。
2. **训练期特权提示（课程脚手架）**：外部教师 LLM 读 rollout 摘要+地图态，产出文本路线提示注入慢塔输入
   （如"你上次向北见过村庄屋顶，本次向北"）。**必须退火**：提示浓度随训练进度→0；
   预注册"脱提示考试"——每周 M-* 评估一律关提示跑，防止提示依赖成瘾。
3. **判优（已建成，v2 强调主判据）**：用户裁定动作正确性的核心判据=**长程动作意图是否明确 +
   是否符合慢塔的文本描述**。即过程优势第二层（意图-行为一致性）升为主判据：慢塔声明的子目标文本
   是行为的"合同"，快头行为偏离合同即扣分、贴合即给分——哪怕最终没成功。里程碑深度分退为骨架，
   LLM judge 抽样审计校准两者。提示与判优共用同一教师上下文，成本合并。

铁律不破：BC 教师必须是学生观测的函数；特权信息只准进奖励塑形与课程提示（且退火），不准进模仿目标。

## 4. 时间估计（单机口径，粗估，可被证伪）

- R0+R1 点火：07-10 前后（已在既定链上）。M-IRON 成功率>0：点火后 1–2 周；两周曲线不动=体系结构性问题（预注册的证伪线）。
- R2a 下界（桶浇铸路线，v2 提前）：M-IRON 稳定后 +1–2 周——解耦钻石后大幅缩短，主要新工作只有
  岩浆池感知类+浇铸宏。
- R2b 钻石：与 R2a 并行，+2–3 周（含地形通道二期与长 episode 税）。
- R4 刷铁机：R2a 后专项 spike 再裁决，不承诺时限。
- 64 核环境农场可把 R2 级以后墙钟压缩约 3–4×。

## 5. 不做什么（边界）

- 不做端到端从零 RL 学合成/放置——GUI 合成闭环 0% 已证伪，宏路线是定案，放置沿用同一教义。
- 不做全图 SLAM/全局规划器——北锚定 clipmap + 慢塔文本规划够用到下界，超出再议。
- 不在 M-* 正式评估里保留任何特权（提示/存档/教师），存档只用于训练课程。
