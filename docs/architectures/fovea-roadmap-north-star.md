# 北极星路线图：多样化成就 · 铁与钻石 · 进入下界

登记日 2026-07-08。终点由用户定义：(A) 多样化成就；(B) 铁与钻石获取；(C) 进入下界。
成功口径沿用 M-IRON 宪法：自然 DEFAULT 世界、随机 seed、冷启动、无教师无特权、结果导向（任何路线均可）、n≥16 报成功率+CI+中位耗时。

## 0. 总判决：可达，但不匀速

三个终点均判**可达**，无需推翻现有体系（双塔+宏路线+GRPO 过程优势）。但三者难度不是线性递增，
而是各自卡在**不同类型**的缺口上：铁卡在长程探索（RL 主战场）；钻石卡在地形/负空间感知（已知 FAIL 项）；
下界卡在**方块放置**这一从未训练过的全新动作域。单机墙钟是全程的税：episode 越深吞吐越低（环境瓶颈机器）。

## 1. 里程碑阶梯（每级 = 能力 + 预注册门）

| 阶段 | 里程碑 | 新增能力 | 门 | 依赖的已知缺口 |
|---|---|---|---|---|
| R0（进行中） | M-WOOD 原木入包 | wood 感知类 + 采木技能 + 木镐宏链 | GRPO 启动门（组内方差>0 / 采木≥0.25） | pos 产率（修采集器中） |
| R1 | M-IRON 铁入包 | GRPO 长程 RL；探索/下洞/村庄任一路线 | M-IRON 成功率>0 且周环比上升 | 村庄/铁傀儡感知类（按需拉动）；MapQuery 0.27 |
| R1.5 | M-STONE→铁工具链 | 熔炉宏+燃料+熔炼宏（原矿→锭）；铁镐宏链 | 宏链端到端冒烟 | 纯宏，低风险 |
| R2 | M-DIAMOND 钻石入包 | 阶梯式下挖宏（Y≈-58 确定性路径）+ 钻石感知类 + 岩浆急停反射 | 同 M-IRON 口径；episode 预算↑至 6000–10000 步 | **Y2 地形 FAIL（负空间 0.266/部署 0.401）→ 深度/几何通道二期**；黑暗视觉 |
| R3 | M-NETHER 跨入下界 | 方块放置技能（全新动作域）+ 黑曜石开采 + 打火石宏链；备选=废弃传送门机会主义路线 | 跨门即成功（不要求存活） | 放置精度（raycast 瞄准+确认回读）；岩浆近距感知 |
| 贯穿 | M-ACH 多样化成就 | GRPO 里程碑深度分扩展为成就集合 + AD 式多样性奖励 | 成就数曲线单调上升 | 低风险（Crafter PPO+AD 已验证同构方案） |

路线要点：
- **钻石不走洞穴探索主路线**，走"阶梯下挖+条带采掘"——把感知-决策难题压缩成宏+矿石识别，
  唯一硬依赖是岩浆急停（挖穿岩浆池是主要死法）。洞穴路线留作 RL 自发发现的加分项。
- **下界主路线**：钻镐→10 黑曜石→摆门→铁+燧石合打火石→点燃。备选：废弃传送门（补 1–2 块黑曜石
  +箱内火焰弹），探索成本换建造成本，两路线都进 GRPO 让学生自选。
- **成就多样化**是主链副产品优先（石器时代/来硬的/热腾腾的/更深的下面等都在必经之路上），
  AD 多样性 bonus 只负责把"顺路"变成"主动收集"。

## 2. 教师角色扩展：判优 + 文本路线提示（用户指令）

教师从纯裁判扩为"裁判+领航员"，三层实现：

1. **塔内提示（推理时合法）**：慢塔本来就是路线提示器——文本计划→向量/索引下行喂快头（§6.2 契约）。
   强化方向：配方卡式提示模板（E2 已验证 0.73）+ 基于地图模块的"未探索扇区"指路。这部分**不违反冷启动**，
   慢塔是学生身体的一部分。
2. **训练期特权提示（课程脚手架）**：外部教师 LLM 读 rollout 摘要+地图态，产出文本路线提示注入慢塔输入
   （如"你上次向北见过村庄屋顶，本次向北"）。**必须退火**：提示浓度随训练进度→0；
   预注册"脱提示考试"——每周 M-* 评估一律关提示跑，防止提示依赖成瘾。
3. **判优（已建成）**：三层过程优势（里程碑深度/意图一致性/全败组条款）+ LLM judge 抽样审计。
   提示与判优共用同一教师上下文，成本合并。

铁律不破：BC 教师必须是学生观测的函数；特权信息只准进奖励塑形与课程提示（且退火），不准进模仿目标。

## 3. 风险登记（诚实清单）

| 风险 | 打击面 | 缓解 | 残余 |
|---|---|---|---|
| 地形/负空间感知（Y2 FAIL 在案） | 钻石（岩浆/坑洞）、下界（岩浆海） | 深度/几何通道二期；近距岩浆=高对比纹理，先训"岩浆急停"专用反射类 | 中。若通道二期不成，钻石改纯宏保守下挖（慢但稳） |
| 方块放置从未训练 | 下界摆门 | 放置=宏优先（raycast 瞄准+方块回读确认），不指望 RL 学会精细放置 | 中低。宏路线确定性高 |
| 黑暗（夜晚/洞内/深层） | 钻石、下界 | 火把宏；或 CraftGround 侧 gamma/亮度设置（待验证）；训练分布纳入暗帧 | 中 |
| 长 episode 吞吐（环境瓶颈机） | R2 起全部 | 存档课程库（中途态起步，Go-Explore 式）省去重复前段；甜点=64 核农场 | 高——单机下 R2/R3 墙钟主要被它拉长 |
| 提示依赖成瘾 | 教师提示机制 | 脱提示考试预注册（每周） | 低 |
| MapQuery strict 0.27 | 长程回航/指路 | 聚类+top-k 在案 | 低 |

## 4. 时间估计（单机口径，粗估可打脸）

- R0+R1 点火：07-10 前后（已在既定链上）。M-IRON 成功率>0：点火后 1–2 周；两周曲线不动=体系结构性问题（预注册的打脸线）。
- R2 钻石：M-IRON 稳定后 +2–3 周（含地形通道二期与长 episode 税）。
- R3 下界：+2–4 周（放置宏+黑曜石链；若废弃传送门路线被 RL 发现可大幅提前）。
- 64 核环境农场可把 R2/R3 墙钟压缩约 3–4×。

## 5. 不做什么（边界）

- 不做端到端从零 RL 学合成/放置——GUI 合成闭环 0% 已证伪，宏路线是定案，放置沿用同一教义。
- 不做全图 SLAM/全局规划器——北锚定 clipmap + 慢塔文本规划够用到下界，超出再议。
- 不在 M-* 正式评估里保留任何特权（提示/存档/教师），存档只用于训练课程。
