# 结论:gaming500 三闸门小规模探针(2026-07-03)

> 对应 2026-07-03 三轮辩论收敛的开工前置验收(闸门条款见当日辩论综合;脚本
> `tests/probe_g500_extract.py` / `probe_g500_gate0*.py` / `probe_g500_gate12.py`,
> 原始指标 `runs/g500_gates/*.json`)。规模:6 游戏 7 段 × 2200 帧 @176px,冻结 DINOv3
> ViT-S/16;动力学用 4×4 池化 + PCA-256 的**降维代理**(非全 121 token,见 §4 关键限定)。

## 1. Gate 0 — Δz→dx 线性可解码性:线性读出 FAIL,信息存在 PASS

- ridge(Δz 全局+行+列均值描述子)→dx:鼠标类各游戏 R² ≈ 0(-0.15~0.00),
  与 shuffle 对照无异;运动分层(p50/p90)不改善;MLP 合池探针同样为负。
- 但诊断三件套证明**不是对齐 bug、信息在**:
  - D1: corr(|dx|, ‖Δz‖) = 0.24~0.36(幅度可见);
  - D2: 错位相关峰在 offset=0(对齐正确;alan-wake 峰偏 +1~+2,疑似相机平滑延迟);
  - D3: patch 网格互相关平移估计与 dx 相关 0.27~0.45(fortnite 高运动 0.45)——
    **方向信息存在于完整 patch 网格的空间结构里,池化/线性读出丢失它**。
- key_attack 线性 AUC 0.647(弱可见)。terraria(2D 光标)与 ghost-of-tsushima
  (dx 全零,疑手柄局)如预期为对照组。

## 2. Gate 1 — 三臂 IG 对照:PASS(弱但方向一致)

同预算三臂(无动作/聚合动作/30Hz 子帧有序动作),1624 train / 273 holdout 窗:

| 口径 | IG(sub−none) | IG(agg−none) |
|---|---|---|
| teacher-forced EV | **+0.034** | +0.0215 |
| 开环 EV@k1-4 | **+0.0363** | — |

- **IG>0 且子帧 > 聚合**——辩论 P13(子帧有序动作条件)获小规模支持;
  在案的"IG 恒负"在此动作编码(逐游戏归一 + mulaw + 子帧)下未复现。
- 限定:单种子无 CI(辩论条款要求跨种子,正式版必须补);逐游戏方向一致但
  palworld 为负;绝对 EV 为负(微型模型+降维代理欠拟合,只有 aim-lab 为正)。

## 3. Gate 2 — CAD 探针头:hard-negative 与深度校准 FAIL(反方预言应验)

- C_act AUC:easy(shuffle)=0.611,**flip(dx 变号)=0.512、swap(子帧对调)=0.523
  ≈随机**——辩论 S2-3"shuffle 负样本只教会幅度匹配、方向绑定不可见"精确应验。
- C_real:real=0.92 vs 生成 k1=0.055——一步生成即与负样本完全可分(负样本太易),
  分数-深度曲线 k1 后**平坦不单调**(S2-2 验收曲线不达标);分数-误差
  Spearman=-0.179(方向对——高分↔低误差,但太弱)。
- D 头:holdout 内时序前 70% 拟合、后 30% 评,Spearman=**-0.363(方向反)**——
  时间分布漂移下误差回归探针失效,单判别头 OOD 不可靠的论点获证。

## 4. 判读与对主线的约束

1. **承重墙部分成立、部分解除**:动作**幅度**在冻结 DINO latent 可见;**方向**只存在
   于完整 patch 网格空间结构(D3),被 4×4 池化+PCA 压掉。⇒ 正式动力学**必须在全
   121 token 网格上做**(dreamer4 SpaceTimeTransformer 本就如此),禁止池化降维捷径;
   本探针的 Gate 2 失败可能是代理压缩伪影,须在全 token 空间复测后才可定罪表征。
2. **子帧动作条件立即采纳**(IG 子帧>聚合>无,双口径一致),按辩论条款进
   dataset.py 契约(变长 padding≤3 + 子帧 mask)。
3. **C_real 负样本必须上难度**(与 rollout 同配置多步生成、混合平稳负样本),
   否则一步饱和、深度校准失效——不确定性来源保留"K 头集成分歧"备胎。
4. IG 闸门正式版:全 token 动力学 + 跨种子 CI + 逐游戏分层(palworld 为负要查)。
5. 数据事实补充:aim-lab 段混入 Apple Music 切换等桌面污染;alan-wake 相机平滑
   使动作-效果延迟 1-2 图像帧(动作条件窗口应含前一区间)。
